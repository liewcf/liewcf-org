## Overview
Frens Edge Switch lets you send a page from Chrome to Edge in a click, ideal for cross-browser testing or Edge-only workflows.

## Highlights
- Open the current tab or context menu links in Edge.
- Lightweight UI and fast actions.
- Handy for QA and compatibility checks.
